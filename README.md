## 🔄 RÉINITIALISER LES DONNÉES

Pour revenir aux 10 pièces de démonstration :

1. Ouvrir l'application
2. Appuyer sur **F12**
3. Dans la Console, taper :
   ```javascript
   localStorage.clear()
   ```
4. Recharger la page (**F5**)

➡️ Les 10 pièces d'origine réapparaissent !

## ❓ PROBLÈMES FRÉQUENTS

### "npm n'est pas reconnu"
➡️ **Solution** : Installez Node.js depuis https://nodejs.org

### Le navigateur ne s'ouvre pas
➡️ **Solution** : Ouvrez manuellement http://localhost:5173

### "Port déjà utilisé"
➡️ **Solution** : Redémarrez votre ordinateur

### Les pièces ne s'affichent pas
➡️ **Solution** : 
1. Appuyez sur F12
2. Regardez s'il y a des erreurs en rouge
3. Si oui, tapez `localStorage.clear()` puis rechargez (F5)
